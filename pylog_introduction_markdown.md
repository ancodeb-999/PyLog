## ğŸ‡ºğŸ‡¸ English
PyLog is a lightweight system process monitoring tool that tracks **process creation** and **process termination** in real time.

### Features
- Detects when a new process is created
- Monitors when a process exits or terminates
- Prints detailed process information (PID, name, etc.)
- Useful for logging, monitoring, and debugging

### Use Cases
- Ensuring certain applications are launched or closed
- Detecting unexpected or malicious processes
- Assisting developers in debugging
- System behavior analysis
